# 前沿分析模块（AHP）测试用例清单

> 适用接口：`POST /api/tasks/{taskId}/ahp-analysis`
>
> 目标：在“暂无真实业务数据”阶段，先用可控的合成数据覆盖核心逻辑、边界输入和异常路径。

---

## TC-01 正常路径：两指标两方案（含 MAX + MIN）

- **目的**：验证标准评分、归一化、排序。
- **输入要点**：
    - 指标：性能(MAX)、成本(MIN)，同一父指标“技术性”。
    - 方案A：性能80、成本30、专家评分8
    - 方案B：性能100、成本20、专家评分5
    - 不传主/次级矩阵（触发等权重）
- **预期**：
    - 指标权重各 0.5
    - 方案B 排名第1，方案A 排名第2

## TC-02 异常路径：任务无指标

- **目的**：验证空指标保护。
- **输入要点**：`taskParameters = []`
- **预期**：抛出 `任务未配置评价指标`

## TC-03 异常路径：任务无候选方案

- **目的**：验证空方案保护。
- **输入要点**：`decisions = []`
- **预期**：抛出 `任务下不存在候选技术`

## TC-04 异常路径：一级矩阵维度不匹配

- **目的**：验证矩阵维度校验。
- **输入要点**：
    - 实际一级指标数量 = 2
    - primaryMatrix 传入 1x1
- **预期**：抛出 `一级指标判断矩阵维度与一级指标数量不一致`

## TC-05 正常路径：二级矩阵缺失走等权重

- **目的**：验证 secondaryMatrices 缺省回退策略。
- **输入要点**：
    - primaryMatrix 正常
    - secondaryMatrices 不传或空
- **预期**：每个父指标下子指标等权分配

## TC-06 正常路径：有二级矩阵时按矩阵计算

- **目的**：验证 secondaryMatrices 生效。
- **输入要点**：
    - 对父指标“技术性”传入 2x2 判断矩阵：[[1,3],[1/3,1]]
- **预期**：第一子指标权重大于第二子指标

## TC-07 边界：单一级指标

- **目的**：验证 `primarySize == 1` 分支。
- **输入要点**：
    - 只有一个 parentCriterion
- **预期**：一级权重固定为 1.0，CI=0，CR=0

## TC-08 边界：判断矩阵中 null/<=0 值

- **目的**：验证矩阵非法值替换逻辑。
- **输入要点**：矩阵含 `null`、`0`、`-1`
- **预期**：这些值被按 1.0 处理，流程不中断

## TC-09 边界：方案缺少某指标值

- **目的**：验证缺失 DecisionParameter 的默认值逻辑。
- **输入要点**：某方案不提供某 TaskParameter 的值
- **预期**：该值按 0.0 参与计算，接口可返回结果

## TC-10 边界：专家评分缺失

- **目的**：验证无专家评分时的总分计算。
- **输入要点**：方案 `expertEvaluations` 为空
- **预期**：expertScore = 0，总分 = ahpScore * 0.8

## TC-11 稳定性：同分并列场景

- **目的**：验证同分情况下排序稳定性（至少不报错）。
- **输入要点**：两方案输入完全相同
- **预期**：都可返回且 rank 连续（1,2）

## TC-12 异常路径：非方阵

- **目的**：验证矩阵必须方阵。
- **输入要点**：传入 `[[1,2],[3]]`
- **预期**：抛出 `判断矩阵必须是方阵`
